Игра на двух игроков.
Один игрок загадывает число, второй отгадывает. Необходимо записать все попытки второго игрока.


Найдите сумму всех элементов в списке.
lst = [34, 2, 342, 90, 1.25, 5.55]

Посчитайте количество отрицательных чисел в списке.
lst = [56, -27, -8, 41, -28, 68, -53, -68, -71, -50, -13]

Удалить все повторяющиеся элементы из списка.
lst = [7, 3, 10, 10, 8, 8, 6, 8, 3, 4]


Напишите функцию, которая создает новый список, содержащий только четные числа из исходного списка.
lst = [41, 37, 33, 6, 45, 24, 11, 49, 47, 0, 42, 12]

Измените каждый элемент списка, умножив его на 2.
lst = [1, 4, 0, 2, 16, 10, 8, 17, 5, 15]


Напишите программу, которая будет запрашивать у пользователя цело-
численные значения и сохранять их в виде списка. Индикатором оконча-
ния ввода значений должен служить ноль. Затем программа должна вы-
вести на экран все введенные пользователем числа (кроме нуля) в порядке
возрастания – по одному значению в строке. Используйте для сортировки
либо метод sort, либо функцию sorted.


Напишите программу, которая будет запрашивать у пользователя чис -
ла, пока он не пропустит ввод. Сначала на экран должно быть выведено
среднее значение введенного ряда чисел, после этого друг за другом не-
обходимо вывести список чисел ниже среднего, равных ему (если такие
найдутся) и выше среднего. Каждый список должен предваряться соот -
ветствующим заголовком.

Напишите функцию, которая генерирует список с десятью случайными числами в диапазоне от 0 до 100. Используйте модуль random.
